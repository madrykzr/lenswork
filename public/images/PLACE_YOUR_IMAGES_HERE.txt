# Photography Studio Asset Directory

You can upload your own photography assets directly to this folder (`/public/images/`) via the file explorer.

Once uploaded, you can reference them directly in your code (e.g., in `/src/data.ts`) using the root path:
`/images/your_filename.jpg`
